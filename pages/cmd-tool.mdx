# Using Command Line Tool

<Callout emoji="ðŸ’¡">
  Before running please check that you have installed required requirements
  [listed here](/#requirements)
</Callout>

<h2> Steps to run Command Line Tool </h2>

1. Clone repository/download the source code from [here](link-to-repository)
2. Extract files if downloaded in zip format
3. Install requirements
4. Run MainToolCode.py

```bash
git clone link-to-repository
cd ApproximateComputingTool
pip install -r requirements.txt
python3 MainToolCode.py
```

<Callout emoji="ðŸ’¡">
  Command Line Tool and GUI Tool has same constraints and same output results.
  It is recommended to use [GUI Tool](/gui-tool/running-the-tool) whenever
  possible
</Callout>
